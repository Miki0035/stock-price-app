## <a name="features">🔋 Features</a>

👉 **User Authentication** : allowing users to securely sign in or sign up to their accounts.

👉 **Interactive Stock Charts** : displaying real-time stock data as graphs for easy trend analysis.

👉 **Stock News & Updates** : providing the latest financial news and market insights in one place.

👉 **Smart Stock Search** : enabling users to search for any stock and view detailed performance metrics.

👉 **Daily News Email Reminder** : sending personalized daily summaries of stock news and updates straight to users’ inboxes.

👉 **Watchlist Management** : allowing users to favorite or track specific stocks for quick access and monitoring.

## <a name="quick-start"> 🤸 Quick Start </a>

Follow this steps to setup the project locally on your machine.

**Prerequsites**

Make sure you have the following installed on your machine

- [Git](https://git-scm.com/)
- [Node](https://nodejs.org/en/download)
- [Next.js](https://nextjs.org/docs)

**Cloning the Repository**

```bash
git clone https://github.com/Miki0035/stock-price-app

cd stock-price-app
```

**Installing dependencies**

Run the following command to install all dependencies

```bash
npm install
```

**Environment variables**

In the root of your project

    - create '.env' file
    - add variables similar to the ones located in .env.example

**Running the Project**

Finally run

```bash
npm run dev
```

In your browser go to 'localhost:3000' 👍
